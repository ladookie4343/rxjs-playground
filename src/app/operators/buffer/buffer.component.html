buffer

  result$ = this.docClicks$.pipe(
    buffer(this.buttonClicks$)
  );

Collects values from the past as an array, and emits that array only when another Observable emits. In this example, you can click anywhere on the page and then click the button and then you'll see all the x coordiantes of the clicks.

result: {{result$ | async | json}}

Another cool pattern to use with buffer is to combine it with debounce time. This waits for something to stop happening and then takes all the values and emits them.

  this.docClicks$.pipe(
    buffer(this.docClicks$.pipe(debounceTime(500)))
  );
